# Saatvik Raghuvanshi Portfolio

Static portfolio website for projects, skills, education, achievements, resume, and contact links.

## Run Locally

Open this folder in VS Code and run the task:

```text
Terminal > Run Task > Serve portfolio
```

Then open:

```text
http://localhost:5500
```

You can also run:

```powershell
python -m http.server 5500
```
